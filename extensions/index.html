VS Code Extensions

Lightweight, focused extensions built to solve specific problems without getting in your way.

Don't Git Lost

Step through any file's git history without leaving VS Code. Inline blame, rich commit hover cards, and time-travel navigation.

Folder Projects

Turn your directory structure into a project list. Switch between any project instantly — no manual registration, no stale lists.

TeamDocs

Keep your team's Markdown docs one click away in the Activity Bar, across every project. Browse, search, and preview without switching windows.
